# ExampleApp - Android sample project for SCCP Lab Team

## Point
注目するべきファイルは以下のとおり。

- app/src/main/java/\*
- app/src/main/res/layout/\*
- app/src/main/res/values/strings.xml
- app/src/main/AndroidManifest.xml
- build.gradle

また、ExampleApp内で使っている、Androidアプリ開発でのキーワードは以下のとおり。

- Activity
- Layout
- View
- API Level
- Intent
- R.java
- string.xml
